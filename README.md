# quickmeet
A video chat/meeting webapp using WebRTC and WebSockets (socket.io). Basically a Google Meet clone + a collaborative whiteboard.

## Install options:

change the default config through these environment variables:
```
PORT = 3000
AUTH = false
AUTH_USER =  user
AUTH_PASS = letsmeet
```
